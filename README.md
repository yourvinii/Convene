# Convene

Convene — MERN + WebRTC based real-time video conferencing app.

## Features
- Video & audio calling
- Real-time chat
- Screen sharing
- Authentication (login/signup)

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: WebRTC, Socket.io

## Setup

### 1. Clone repo
git clone https://github.com/your-username/convene.git

### 2. Install dependencies
cd client && npm install  
cd ../server && npm install  

### 3. Run app
cd server && npm start  
cd client && npm start
